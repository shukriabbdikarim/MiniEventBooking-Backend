# Clean Architecture Web API

A simple ASP.NET Core Web API built with Clean Architecture.

## Technologies

* ASP.NET Core 8
* Entity Framework Core
* SQLite
* MediatR
* Swagger

## Features

* Product CRUD
* Product and Category entities
* Repository Pattern
* CQRS with Commands and Queries





\# Clean Architecture Web API

\- API
